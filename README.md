# car2Park_server_2

To put server to sleep, enter the cmd

>heroku ps:scale web = 0

in the directory car2Park_server_2

***

To wake_up server, enter the cmd

> heroku ps:scale web = 1

in the directory car2Park_server_2

***

To push code to heroku

> git push heroku master

in the directory car2Park_server_2
